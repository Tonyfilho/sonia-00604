UC00604 – Programar para a web (Frontend)

Código: UC00604 | Créditos: 4.5

Descrição Geral

Esta unidade curricular foca o desenvolvimento de competências em programação web no lado do cliente (frontend), incluindo HTML, CSS, JavaScript e princípios de acessibilidade e responsividade.

Objetivos

  • Estruturar páginas web com semântica correta.
  • Atribuir estilos utilizando CSS.
  • Programar interatividade com JavaScript básico.
  • Aplicar práticas de acessibilidade.

Principais Conteúdos

  • Estrutura de páginas com HTML
  • Estilos com CSS
  • Layouts responsivos
  • Formulários e tabelas
  • JavaScript no cliente
